# SOCIAL NETWORK PARSER

## 1) DOWNLOAD FILE:
```bash
- main.py
```

## 2) CREATE FILE "config.ini":
```bash
[DEFAULT]
TELEGRAM_TOKEN = {YOUR BOT TOKEN}
```

## 3) STARTING:
```bash
- run main.py
- run Telegram app
- search for @fp_chatbot
- send "/start" or "/help" to bot to get all commands
- send commands to bot to get information
```
